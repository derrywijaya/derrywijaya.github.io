Bilingual Lexicon Induction


Description: Induce word translations from monolingual corpora in many languages. Machine translations are typically learned from parallel corpora. However, for many language pairs, such parallel corpora cannot be assumed. In such cases, being able to learn translations from monolingual sources is potentially useful.

Papers:
  • Learning Translations via Matrix Completion (EMNLP 2017) (pdf)(code)

Output: Translations from foreign words to English, for all words occurring >100 times in Wikipedia (showing top-10 English translations for each word)
